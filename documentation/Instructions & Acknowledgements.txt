INSTRUCTIONS
In order to run the game, open the file Game\AirHockey.sln with Microsoft Visual Studio 2008.
Once open, click on the Start Debugging (F5) button.

If running a multiplayer game then you should ensure an instance of AirHockeyServer.exe is 
running.
